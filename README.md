# 🚌 버스 시간표

버스 시간표를 쉽게 조회할 수 있는 웹 앱입니다.

## 기능

- ✅ 조건 선택 (평일/주말/방학, 왕편/복편)
- ✅ 전체 시간표 표시
- ✅ 지금 출발 가능한 버스만 보기
- ✅ 현재 시간 ±3시간만 보기
- ✅ 즐겨찾기 저장
- ✅ 가장 가까운 출발 시간 하이라이트

## 사용법

1. 웹 서버로 실행하거나 `index.html`을 브라우저에서 열기
2. 상단에서 날짜 유형(평일/주말/방학)과 방향(왕편/복편) 선택
3. 필요에 따라 필터 옵션 체크
4. 원하는 시간에 ★ 버튼을 눌러 즐겨찾기 저장

## 시간표 수정

`data/schedules.json` 파일을 수정하면 시간표를 업데이트할 수 있습니다.

## 프로젝트 구조

```
bus/
├── index.html          # 메인 HTML
├── css/
│   └── style.css       # 스타일
├── js/
│   ├── app.js          # 메인 로직
│   ├── schedule.js     # 시간표 처리
│   ├── filter.js       # 필터링
│   ├── favorites.js    # 즐겨찾기
│   └── utils.js        # 유틸리티
├── data/
│   └── schedules.json  # 시간표 데이터
└── docs/
    └── PLANNING.md     # 기획서
```
